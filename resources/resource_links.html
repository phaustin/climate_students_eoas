Working with model datasets#

CMIP6#

  • Lamont course on earth data – 4 notebooks on CMIP6 model analysis

  • CMIP6 scenarios – model forcing scenarios (shared socio-economic pathways)

  • IPCC working group 1 sixth assessment report

  • Processed time series and maps

  • google sheet with CMIP6 variables

  • Project Pythia CMIP6 cookbook – workflow examples for climate senstivity, mean surface temperature, precipitation, regridding, multimodel means

  • Using the Pangeo catalog – how to search and load datasets - includes CMIP6 and LENS (large ensembles), has a variety of MIP experiments, including the standard ‘scenario’ experiments, but also 1%CO2 increase experiments, land-use experiments (including deforestation), polar-amplification experiments….

  • CMIP6 variable list – all the model variables in the CMIP6 table

  • Other CMIP6 data access – where to download data from model experiments that aren’t included in the Pangeo catalog, including other MIPS

  • Model Intercomparison Projects (MIPs) included in CMIP6 – details on difference simulations run by multiple climate modelling centres across the World to address apecific questions about the climate

  • CESM2-LENS Details of the CESM2 Large ENsemble Simulations (LENS) that can be used to address questions of natural variability.

  • CESM1-LENS data and example jupyter notebooks – some examples of jupyter notebooks analysing large ensemble data

  • Climate Data Gateway at NCAR – Additional access to CESM simulations, including the CESM2-LENS

  • Introduction to Taylor diagrams – a tool for comparing models against each other and against data
